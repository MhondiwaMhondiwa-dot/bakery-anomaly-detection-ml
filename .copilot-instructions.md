# Copilot Instructions

This repository uses a special file to communicate persistent instructions for GitHub Copilot and other automated assistants.

## Current Rules

- **Dispatch EDA**: Always use `src/analysis/eda_dispatch_enhanced.py` when performing exploratory data analysis on the dispatch dataset. The legacy `eda_dispatch.py` script exists for backward compatibility, but it should not be run against the current processed data schema (depot‑centric, multi‑SKU). 
- **General rule**: prefer `*_enhanced.py` versions of EDA scripts wherever they exist; they target the latest schema and include additional sanity checks/visuals. Legacy variants remain for reference only.
- Additional instructions will be added to this file as needed.
